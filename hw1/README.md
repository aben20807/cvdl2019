# cvdl2019 hw1

## Usage

```bash
$ env/bin/python3 app.py
```

## Python packages

```bash
$ virtualenv env
$ env/bin/pip3 install -r requirements.txt
$ env/bin/pip3 install torch==1.3.0+cpu torchvision==0.4.1+cpu -f https://download.pytorch.org/whl/torch_stable.html
```
